CCS Technology — Downloads
==========================

Place distributable binaries in this folder. They are served at
https://www.ccs-technology.com/downloads/<filename>

Expected files (referenced by src/lib/site-data.js -> DOWNLOADS):
  - atheer-android.apk   (Atheer for Android)

Until the real APK is placed here, the Android download button on
/downloads will 404. Either drop the file in this folder, or set
`available: false` for the Android entry in src/lib/site-data.js to
show a "Coming Soon" state instead.
